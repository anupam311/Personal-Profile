## Overview

This repository contains the source code for my **personal developer portfolio website**.

The portfolio showcases my **projects, technical skills, and development journey** as a **BTech student specializing in Artificial Intelligence and Machine Learning**.

It acts as my **online resume**, highlighting my work in **web development, backend development, and software engineering**.

The website is designed with a clean UI and responsive layout to ensure a smooth experience across desktop and mobile devices.

## Running the Project Locally

To run the portfolio website locally:

1. Clone the repository

```
git clone https://github.com/anupam311/Personal-Profile.git
```

2. Open the project folder

3. Open **index.html** in your web browser

No additional setup is required.
